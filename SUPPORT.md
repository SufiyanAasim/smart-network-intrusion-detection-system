# Support

- **Bugs / feature requests:** open a GitHub issue using the templates in
  `.github/ISSUE_TEMPLATE/`.
- **Security issues:** see [SECURITY.md](SECURITY.md) — do not file a public issue.
- **Usage questions:** check [docs/guides/](docs/guides/) and
  [docs/troubleshooting/troubleshooting.md](docs/troubleshooting/troubleshooting.md) first.
