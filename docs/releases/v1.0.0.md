## Highlights
First structured pre-release: the working Streamlit IDS is now a
professionally laid-out repository with tests, CI, and docs, ready to
iterate toward a stable v1.0.0.

## Added
- Repo restructure: `src/`, `data/`, `models/`, `notebooks/`, `config/`, `docs/`, `scripts/`, `tests/`.
- `src/nids/features.py` — feature engineering extracted from `app.py`, unit tested.
- `tests/test_features.py` — pytest suite for feature logic.
- `scripts/train_models.py` — CLI retraining entrypoint.
- `.github/` — lint/test workflows, issue templates, PR template, CODEOWNERS, dependabot.
- `Dockerfile`, `docker-compose.yml`, `Makefile`.
- README, CHANGELOG, CONTRIBUTING, SECURITY, CODE_OF_CONDUCT, SUPPORT, RELEASE, ROADMAP.

## Changed
- `app.py` path resolution updated for the new `models/` and `data/nsl-kdd/` locations.

- N/A (first structured release)

  fields rather than computing full windowed stats (see ROADMAP.md).
