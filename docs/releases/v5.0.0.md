# 🚀 NIDS v5.0.0

Codename: Bulwark

## Release date
2026-07-17

## Highlights
Makes the persisted detection history genuinely useful to work with: export
all of it, and drill into any single source IP.

## Added
- Full history export — download the entire persisted `data/history.db` as
  CSV or Excel (`.xlsx`) from the History tab, not just the visible 200-row
  window (`storage.query_all()`). The Excel button is hidden gracefully if
  `openpyxl` isn't installed.
- Per-IP drill-down — pick a source IP on the History tab to see every past
  detection for it across sessions, with a per-IP summary (total detections,
  RF/DT attack counts, first/last seen). Backed by `storage.query_by_ip()`,
  `storage.query_ip_summary()`, and `storage.query_distinct_ips()`.

## Changed
- Migrated all `use_container_width=True` calls to `width='stretch'`,
  clearing Streamlit's deprecation warnings from the server logs.

## Fixed
- N/A

## Known issues
- Excel export requires `openpyxl` (now in `requirements.txt`); without it,
  only CSV export is offered.

## Compatibility
- Requires Python 3.11+.
- New dependency: `openpyxl>=3.1`. Run `pip install -r requirements.txt`
  after upgrading. SQLite schema is unchanged — existing `history.db` works.

## Upgrade instructions
`pip install -r requirements.txt` (for openpyxl), then restart the app.
