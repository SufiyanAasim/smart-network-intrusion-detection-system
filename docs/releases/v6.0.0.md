# 🚀 NIDS v6.0.0

Codename: Aegis (grand release — 5 features)

## Release date
2026-07-17

## Highlights
The biggest release so far: geographic context, richer reporting, live
throughput, richer alerting, and capture-readiness detection.

## Added
- **GeoIP + source-IP geography** (`src/nids/geo.py`) — History tab shows a
  breakdown of distinct source IPs by type (private / public / loopback /
  reserved). With `geoip2` + a MaxMind GeoLite2-City DB (`GEOIP_DB_PATH`),
  public IPs are plotted on a world map; otherwise a caption explains how to
  enable it. Stdlib-only classification always works.
- **PDF report export** (`src/nids/reporting.py`) — "📄 Download PDF report"
  beside the CSV download, summarizing totals, per-model attack counts, and
  top attacker IPs (reportlab).
- **Real-time throughput graph** (`src/nids/throughput.py`) — live
  packets/sec + KB/sec area chart over a rolling 60-second window in the
  Live Capture tab.
- **Sound / browser alert notification** (`src/nids/notify.py`) — opt-in
  sidebar toggles play a synthesized beep and/or raise a desktop
  Notification on a critical threat, cooldown-throttled with existing alerts.
- **Npcap install-check banner** (`src/nids/netcheck.py`) — detects a
  missing packet-capture provider and shows a platform-specific fix
  (Npcap download link on Windows), disabling "Start Capture" instead of
  silently capturing nothing.

## Changed
- Live Capture tab now shows a throughput chart above the results and a
  capture-readiness banner.
- Results downloads now offer PDF alongside CSV.

## Known issues
- The GeoIP world map needs `geoip2` **and** a MaxMind `.mmdb` file
  (`GEOIP_DB_PATH`); without them the IP-type breakdown is still shown.
- Sound/notification and the live throughput chart require an actual live
  capture, which needs a working Npcap/libpcap provider.

## Compatibility
- Requires Python 3.11+.
- New dependencies: `reportlab>=4.0` (PDF export), `geoip2>=4.7` (optional
  GeoIP). Run `pip install -r requirements.txt` after upgrading. SQLite
  schema unchanged.

## Upgrade instructions
`pip install -r requirements.txt`, then restart the app. Optionally set
`GEOIP_DB_PATH` in `.env` to enable the world map.
