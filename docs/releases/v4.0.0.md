# 🚀 NIDS v4.0.0

Codename: Citadel

## Release date
2026-07-16

## Highlights
First release under the locked 2-feature cadence: adjustable alert
sensitivity and a proper trend view for detection history.

## Added
- Configurable CRITICAL threshold — sidebar slider (`⚙️ Thresholds`,
  default 20%, `CRITICAL_THRESHOLD_PCT` env default) replaces the hardcoded
  cutoff in `generate_smart_summary`. SUSPICIOUS remains anything above 0%
  and below the CRITICAL threshold.
- History tab "Attacks over time" trend chart (RF vs. DT, per-minute
  buckets), via new `storage.query_trend()`.
